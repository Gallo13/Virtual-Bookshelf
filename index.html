Purpose and Functionality

This Virtual Bookshelf serves as a way
to store all the books I have read
and own. The technologies used
to make this website are Python,
MySQL, and Flask. This website
allows me to update my database.
This project is scalable and I
will add data analytics to query
the data and learn more about
my reading habits. In the future,
future, I will also add a recommender
system to find more books that would
suit my taste.
